# Tarefas - Fundamentos de Node.js

Este repositório faz parte do curso de **Node.js** da Rocketseat.

## Sobre

Estou começando a trilha de Node.js de novo.

Não é exatamente começar do zero como se eu nunca tivesse visto nada, mas é voltar com mais calma, prestando atenção nos detalhes que talvez eu tenha deixado passar antes. Aos 21 anos, às vezes bate aquela sensação de que eu já deveria saber mais, estar mais avançado ou ter tudo mais organizado. Mas também estou entendendo que revisar, recomeçar e reforçar a base faz parte do processo.

Quero fazer essa jornada sem depender de IA ou Google para resolver as tarefas. A ideia é usar a documentação, quebrar a cabeça quando precisar e realmente entender o que estou escrevendo.

## Objetivo

Este repositório vai servir como um registro das tarefas que eu for fazendo ao longo dos módulos.

Mais do que só entregar exercícios, quero guardar aqui um pouco da minha evolução: o que eu sabia, o que fui aprendendo, onde travei e como fui melhorando com o tempo. Também quero respeitar o momento de cada módulo, sem tentar colocar coisa avançada onde ainda não faz sentido. Se a tarefa é só entender como ler um arquivo, não faz sentido transformar isso em uma API cheia de camadas só para parecer mais completo. A ideia é deixar o aprendizado acontecer de forma natural, etapa por etapa.
